Лукьяненко Александр, 302 группа.

Реализованные эффекты:
	1. Стандартная модель освещения (Фонга);
	2. Слоистый туман (вершинный + добавлена зависимость от глубины);
	3. Normal Mapping (см. на паука, 
	   карта нормалей: bmp/normal.bmp,
	   цвет паука: bmp/color.bmp); 
	4. Нестандартная модель освещения (Кук-Торренс,
	   см. шарик);
	5. Нечеткие тени (карта теней);

Инструкция по сборке:
В bin находится папка "библиотеки".
Как подключить библиотеки:

1. Каталоги файлов
Открыть проект в Visual Studio->Проект->Свойства Проекта->
C/C++->Общие->Дополнительный каталоги включаемых файлов->
(Прописыаем маршруты:
"...\MyGraphic\bin\Библиотеки\glm"
"...\MyGraphic\bin\Библиотеки\include\glaux"
"...\MyGraphic\bin\Библиотеки\include\glew"
"...\MyGraphic\bin\Библиотеки\include\glfw"
"...\MyGraphic\bin\Библиотеки\include\glut\GL")

2. Каталоги библиотек:
Компоновщик->Дополнительные каталоги библиотек->
(Прописываем маршруты:
...\MyGraphic\bin\Библиотеки\lib\glut
...\MyGraphic\bin\Библиотеки\lib\glfw\lib-vc2017"
...\MyGraphic\bin\Библиотеки\lib\glew
...\MyGraphic\bin\Библиотеки\lib\glaux)

3. Зависимости:
Компоновщик->Ввод->Дополнительные зависимости->
(Прописваем:
glfw3.lib
glew32s.lib
glaux.lib
glut32.lib)


Для перемещения по сцене использовать клавиши: A,D,S,W
Чтобы включить/отключить туман, использовать клавишу: F/G
Чтобы закрыть окно, использовать клавишу: ESC
