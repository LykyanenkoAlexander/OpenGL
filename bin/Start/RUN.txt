Правила эксплуатации:

Для перемещения по сцене использовать клавиши: A,D,S,W
Включить туман: F
Отключить туман: G
Можно использовать компьютерную мышку
Чтобы закрыть окно, использовать клавишу: ESC

Прочее:
Шейдеры расположены в папке Start/shaders
Используемые изображения расположены в папке Start/bmp
Используемые модели расположени в папке Start/obj

Дополнительно:
Скриншоты сцены находятся в папке Start/My_scene
